# Nebula
Vaugely balanced mod for Balatro. Originally intended as art practice (it might be more than that now)

Not ready for a proper release yet, only about 40% of the cards are finished (art is pretty much finished though)

Comes with a bit over 60 jokers and a new Dice consumable set + related items (the latter is unimplimented currently)
Generally attempted to add more jokers that encourage playing hands other than Pair, though there are exceptions
Also a number of effects related to Debuffed Cards, mostly jokers that are strong for their rarity in exchange for the effect of the card not activating
